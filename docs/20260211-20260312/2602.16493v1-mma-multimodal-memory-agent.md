## 摘要
长程多模态智能体依赖于外部记忆；然而，基于相似性的检索往往会提取出陈旧、低可信度或冲突的项，这可能引发过度自信的错误。我们提出了多模态记忆智能体（MMA），它通过结合来源可信度、时间衰减和冲突感知的网络共识，为每个检索到的记忆项分配动态可靠性评分，并利用该信号重新加权证据，在支持不足时选择弃权。我们还引入了 MMA-Bench，这是一个通过程序生成的基准测试，用于评估具有受控说话者可靠性和结构化文本-视觉矛盾的信念动力学。利用该框架，我们揭示了“视觉安慰剂效应”（Visual Placebo Effect），展示了基于检索增强生成（RAG）的智能体如何从基础模型中继承潜在的视觉偏见。在 FEVER 数据集上，MMA 在保持与基线相当的准确率的同时，将方差降低了 35.2% 并提高了选择性效用；在 LoCoMo 上，面向安全的配置提高了可执行准确率并减少了错误回答；在 MMA-Bench 上，MMA 在视觉模式下达到了 41.18% 的 Type-B 准确率，而基线在相同协议下则崩溃至 0.0%。代码：https://github.com/AIGeeksGroup/MMA。

## Abstract
Long-horizon multimodal agents depend on external memory; however, similarity-based retrieval often surfaces stale, low-credibility, or conflicting items, which can trigger overconfident errors. We propose Multimodal Memory Agent (MMA), which assigns each retrieved memory item a dynamic reliability score by combining source credibility, temporal decay, and conflict-aware network consensus, and uses this signal to reweight evidence and abstain when support is insufficient. We also introduce MMA-Bench, a programmatically generated benchmark for belief dynamics with controlled speaker reliability and structured text-vision contradictions. Using this framework, we uncover the "Visual Placebo Effect", revealing how RAG-based agents inherit latent visual biases from foundation models. On FEVER, MMA matches baseline accuracy while reducing variance by 35.2% and improving selective utility; on LoCoMo, a safety-oriented configuration improves actionable accuracy and reduces wrong answers; on MMA-Bench, MMA reaches 41.18% Type-B accuracy in Vision mode, while the baseline collapses to 0.0% under the same protocol. *   **研究动机：** 现有的长程多模态智能体（Agents）高度依赖外部记忆检索（RAG）。然而，传统的基于“向量相似度”的检索存在严重缺陷：它无法区分陈旧、低可信度或相互冲突的信息。这导致智能体在面对噪声输入时，容易产生“过度自信的错误”（Over-confident errors），即在证据不足或矛盾时仍给出错误的肯定回答。
*   **核心问题：** 如何在记忆检索层面建立一套可靠性评估机制，使智能体具备“认知审慎”（Epistemic Prudence），能够识别不可靠证据并在必要时选择弃权（Abstain）。

### 2. 方法论：MMA 框架
MMA（Multimodal Memory Agent）在基础记忆系统（如 MIRIX）之上增加了一个**元认知可靠性层**。
*   **核心思想：** 为每一个检索到的记忆项 $M_i$ 计算一个动态可靠性评分 $C(M_i)$，用于重新加权证据。
*   **关键技术细节（三个维度）：**
    1.  **来源可靠性 $S(M_i)$：** 根据记忆来源（如不同用户或数据库）预设的信任先验进行打分。
    2.  **时间衰减 $T(M_i)$：** 使用指数衰减函数 $T(M_i) = \exp(-\frac{\ln(2)}{T_{half}} \Delta t_i)$，模拟信息随时间流逝而过时。
    3.  **网络共识 $C_{con}(M_i)$：** 衡量该记忆项在检索到的邻域内是否得到其他记忆的支持。通过计算语义相似度加权的共识分，正向加强一致信息，负向惩罚冲突信息。
*   **决策逻辑：** 最终得分是上述三者的加权和。智能体根据该得分决定是采纳记忆、降低其权重，还是在支持度不足时触发弃权机制。

### 7. 优点（亮点）
*   **多维度可靠性建模：** 不同于仅靠相似度的检索，MMA 综合了来源、时间和共识，更符合人类处理信息的逻辑。
*   **MMA-Bench 的设计：** 该基准通过控制变量（来源先验、视觉冲突）精准诊断了智能体的认知缺陷，具有很高的学术价值。
*   **认知审慎：** 赋予了智能体“知之为知之，不知为不知”的能力，这对安全关键型应用（如医疗、法律助手）至关重要。

### 8. 不足与局限
*   **依赖检索召回：** MMA 是一个后检索处理模块，如果底层的 RAG 系统根本没有检索到相关上下文，MMA 无法凭空修复缺失的信息。
